# install epos

1. Download [gcc-4.4.4](http://epos.lisha.ufsc.br/dl87) and extract in "/usr/local/ia32"

2. Instal libs
	* linux-libc-dev:i386
	* gcc
	* gcc-4.4
	* gcc-4.4-base
	* gcc-multilb
	* gcc-4.4-multilib
	* lib32gcc1
	* lib32gomp1
	* libgcc1
	* libgomp1
	* bin86
	* ia32-libs
	* lib32stdc++6
	* libc6-i386
	* libc6-dev-i386
